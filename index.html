🪟 Windows 代理软件

⚠️ 重要：防止 DNS 泄露（必读）

DNS 泄露会让您的 ISP 看到您访问的所有域名，即使使用了代理！

方法一：软件内设置（推荐）

大多数现代代理客户端都内置了 DNS 防泄露功能：

  • Clash Verge Rev：设置 → 系统设置 → TUN 模式 → 开启
  • v2rayN：设置 → Core:基础设置 → DNS 对象 → 启用防泄露
  • NekoRay：设置 → 核心 → 启用 FakeDNS

方法二：修改系统 DNS

  1. 打开 控制面板 → 网络和共享中心
  2. 点击当前网络连接 → 属性
  3. 选择 Internet 协议版本 4 (TCP/IPv4) → 属性
  4. 选择 使用下面的 DNS 服务器地址：

首选 DNS：8.8.8.8 或 1.1.1.1
备用 DNS：8.8.4.4 或 1.0.0.1

验证是否泄露

访问 dnsleaktest.com 进行测试

⚠️ 如果显示您真实的 ISP 名称和位置，说明存在 DNS 泄露！

⚡ Clash Verge Rev v2.4.7

基于 Clash Meta 内核的现代化 GUI 客户端，使用 Rust + Tauri 开发，性能优异，界面美观，支持 Windows 7/8/10/11 全系统。

📖 详细使用教程

  1. 下载安装：下载 .exe 安装包，双击运行安装
  2. 首次启动：可能会被杀毒软件拦截，选择"允许"即可
  3. 导入订阅：
    • 点击左侧"订阅" → "新建"
    • 粘贴订阅链接 → 点击"导入"
    • 点击"刷新"获取节点
  4. 开启系统代理：
    • 点击左侧"代理" → 选择节点
    • 点击顶部"系统代理"开关
  5. 防止 DNS 泄露（重要！）：
    • 设置 → 系统设置 → TUN 模式 → 开启
    • 设置 → DNS 设置 → 启用"DNS 增强模式"
💡 提示：TUN 模式会创建虚拟网卡，可以接管所有应用的流量，比系统代理更彻底。
⚠️ 注意：Windows 7 用户需要先安装 Universal C Runtime

🔧 v2rayN 7.20.4

经典的 V2Ray Windows 客户端，支持 VMess、VLESS、Trojan、Shadowsocks 等多种协议，功能丰富，适合进阶用户。

📖 详细使用教程

  1. 下载解压：下载 ZIP 文件，解压到任意目录
  2. 运行程序：双击 v2rayN.exe 启动
  3. 添加服务器：
    • 服务器 → 添加 VMess/VLESS 服务器
    • 填入服务器信息（地址、端口、用户ID等）
    • 或扫描屏幕二维码
  4. 导入订阅：
    • 订阅分组 → 订阅分组设置
    • 添加 → 粘贴订阅地址
    • 右键订阅 → 更新订阅
  5. 防止 DNS 泄露：
    • 设置 → Core: 基础设置
    • DNS 对象 → 选择"远程DNS"
    • 勾选"启用 DNS 防泄露"
  6. 启动代理：
    • 选择服务器 → 回车确认
    • 系统代理 → 自动配置系统代理
💡 提示：v2rayN 内置了 Xray 和 v2ray 两个核心，建议使用 Xray 核心（默认），性能更好。

🐧 Linux 代理软件

⚠️ Linux DNS 防泄露设置

修改 /etc/resolv.conf

sudo nano /etc/resolv.conf

添加以下内容：

nameserver 8.8.8.8
nameserver 1.1.1.1

或使用 systemd-resolved

sudo systemctl enable --now systemd-resolved

编辑 /etc/systemd/resolved.conf：

DNS=8.8.8.8 1.1.1.1
