Clash Verge Rev

v2.4.7

基于 Clash Meta 内核的现代化 GUI 客户端，使用 Rust + Tauri 开发，性能优异，界面美观。支持 Windows 7/8/10/11 全系统。

📖 详细使用教程

  1. 下载安装：下载 .exe 安装包，双击运行安装
  2. 首次启动：可能被杀毒软件拦截，选择"允许"
  3. 导入订阅：
    • 点击左侧"订阅" → "新建"
    • 粘贴订阅链接 → 点击"导入"
    • 点击"刷新"获取节点列表
  4. 选择节点：点击左侧"代理" → 选择延迟最低的节点
  5. 开启代理：点击顶部"系统代理"开关
💡 推荐：开启 TUN 模式可以接管所有应用流量，比系统代理更彻底。设置 → 系统设置 → TUN 模式 → 开启
⚠️ Windows 7 用户需先安装 Universal C Runtime

🛡️ DNS 防泄露完整指南

DNS 泄露会让您的 ISP 看到您访问的所有域名，即使使用了代理！以下是完整的防护方案：

方法一：软件内设置（推荐）

  1. Clash Verge Rev：设置 → 系统设置 → TUN 模式 → 开启
  2. v2rayN：设置 → Core:基础设置 → DNS 对象 → 启用防泄露
  3. NekoRay：设置 → 核心 → 启用 FakeDNS

方法二：修改系统 DNS

  1. 打开 控制面板 → 网络和共享中心
  2. 点击当前网络连接 → 属性
  3. 选择 Internet 协议版本 4 (TCP/IPv4) → 属性
  4. 选择 使用下面的 DNS 服务器地址：
    • 首选 DNS：8.8.8.8 或 1.1.1.1
    • 备用 DNS：8.8.4.4 或 1.0.0.1

验证是否泄露

访问 dnsleaktest.com 进行测试

⚠️ 如果显示您真实的 ISP 名称和位置，说明存在 DNS 泄露！
