🪟 Windows 代理软件

⚠️ 重要：防止 DNS 泄露

DNS 泄露会导致您的真实访问记录被 ISP 记录。请按以下步骤设置：

方法一：软件内设置（推荐）
在 Clash Verge Rev 中：
设置 → 网络 → 增强模式 → 选择 TUN 模式
设置 → DNS → 启用 DNS 防泄露
方法二：系统 DNS 设置
控制面板 → 网络和共享中心 → 更改适配器设置
右键网络连接 → 属性 → IPv4 → 使用以下 DNS：
首选：8.8.8.8 或 1.1.1.1
备用：8.8.4.4 或 1.0.0.1
验证 DNS 是否泄露
访问 dnsleaktest.com 进行测试
如果显示的是您真实的 ISP DNS，说明存在泄露！

🐧 Linux 代理软件

⚠️ 重要：防止 DNS 泄露

修改 /etc/resolv.conf
sudo nano /etc/resolv.conf
添加：nameserver 8.8.8.8
nameserver 1.1.1.1
或使用 systemd-resolved
sudo systemctl enable systemd-resolved
编辑 /etc/systemd/resolved.conf 设置 DNS
